How to compile and run code under C programming

2020-07-02
Share
  Tweet  

Go to the terminal

Creat a folder for the task.

Create the filename with extension code .c as follows

vi file_name.c

Once the file is completed, then compile the file as follows

gcc file_name.c -o compiled_file_name

The compiled code will be accessible via compiled_file_name

then run to programm by issuing the following command.

./compiled_file_name
